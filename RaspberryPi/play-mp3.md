MP3の再生
======================================================================

apleyではmp3は再生できないのでパッケージをインストールする
```
sudo apt install mpg321
```

再生
```
mpg321 audio.mp3
```

-qでコンソールの出力なし


音量調整は以下のコマンドでできる
```
alsamixer
```
